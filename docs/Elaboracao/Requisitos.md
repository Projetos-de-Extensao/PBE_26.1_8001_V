# Leventamento de Requisitos e Casos de Uso

**Sistema:** Sistema de Gestão e Validação de Estágios

---

## 1. Perfis de Usuários do Sistema

* **Alunos:** Usuários responsáveis por enviar documentos relacionados ao estágio (contratos, termos e relatórios), além de acompanhar o andamento das solicitações dentro do sistema.
* **Coordenador / Setor de Estágios:** Responsáveis por analisar as solicitações

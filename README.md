# Projeto Back-End 

**Código da Disciplina**: IBM8936<br>

Este projeto consiste no desenvolvimento de uma aplicação back-end voltada para o gerenciamento e validação de documentos de estágio.

A proposta é permitir que estudantes realizem o envio de documentos, acompanhem o andamento das solicitações e que a coordenação consiga analisar, validar e aprovar os processos de forma organizada.

O sistema busca otimizar o fluxo de validação, reduzir erros e centralizar todas as informações em uma única plataforma.
